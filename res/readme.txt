It's important that files in this directory are not deleted!

File Explanation:

	Ace.dat - Called "compiler" in the Visual Ace IDE. Compiled code is added to this
		  file and then saved as an executable to "compile" a windows executable.

	code.brw - Stored code for the code browser.

	proc.vas - Stored procedures and types, the IDE accesses this file.

	Procedures.txt - A bit outdated but still a good start for created procedures.

	*.ico - Icons used when associating file types in the registry.
